# jm-desktop

这是一个禁漫的第三方客户端，基于 electron + vue 构建。

## NSFW 警告⚠

请勿在公共场合使用本软件。

## 功能

- [x] 登录
- [x] 搜索
- [x] 阅读
- [x] 收藏
- [x] 简易用户信息
- [x] 阅读历史
- [x] 漫画详情
- [x] 历史评论
- [x] 漫画评论
- [x] 发表评论
- [x] 相关漫画
- [x] 首页推荐
- [x] 漫画下载（JCoin 付费本目前不支持）
- [x] 目录阅读

## 安装包

只提供 win 下的 zip 安装包，解压即用，所有数据只在解压的目录中，整体文件夹可随意拷贝移动（除了下载路径）。
